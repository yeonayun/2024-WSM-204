# style 적용 3가지 방법
1. <. . . style = " . . . ">
2. `<style>. . . </style>`
3. `<link rel = "stylesheet" href = "css파일.css">`

# 형식
```css
선택자 {
    속성: 값; /* 선언 */
}
```

# 선택자
```
*           : 모든 것
TAG         : HTML TAG
.class      : class 속성
#id         : id 속성

선택자1, 선택자2    : 선택자1과 선택자2
선택자1 선택자2     : 선택자1의 자손 선택자2
선택자 > 선택자2    : 선택자1의 자식 선택자2

선택자1 + 선택자2   : 선택자1 바로 아래 자매
선택자2
선택자1 ~ 선택자2   : 선택자1 아래 모든 자매

```
